##Create a Sticky Footer We'll be following these instructions: http://css-tricks.com/snippets/css/sticky-footer/ First, we need to create a page wrapper around the page content only, not the footer content. The home page HTML will look like similar to this: ``` <div class="page-wrap"> <nav> ... </nav> <header> <h1>Jane Smith</h1> <h2>Web Developer + City Girl</h2> </header> </div><!-- close .page-wrap --> <footer> ... </footer> ``` The about and contact pages will look like this: ``` <div class="page-wrap"> <header> <h1>Jane Smith</h1> <h2>Web Developer + City Girl</h2> <nav> ... </nav> </header> </div><!-- close .page-wrap --> <footer> ... </footer> ```
##Sticky Footer CSS ``` /* sticky footer */ html, body { height: 100%; } .page-wrap { min-height: 100%; margin-bottom: -35px; /* must be the same height as footer */ } .page-wrap:after { content: ""; display: block; } footer, .page-wrap:after { height: 35px; /* wowever tall you want the footer to be */ } footer { background: #222; color: #ccc; } ```
##One more update Let's revise the CSS for the paragraph in the footer. ``` footer { clear: both; padding: 5px; /* adds space inside the footer */ } footer p { margin: 0; /* removes default margin from paragraph */ } ``` Compare your page with this [example](../project/examples/sticky-footer-home.html).
