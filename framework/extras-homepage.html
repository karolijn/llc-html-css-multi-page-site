##Home page styles Before creating the home page drop down menu, we'll need to talk about icon fonts and pseudo class selectors first. ###Icon fonts Icon fonts are an easy way to add imagery to your web page but still have the flexibility of styling properties like size and colour using CSS since they *are* fonts! There's many to choose from but this example uses Font Awesome: [http://fortawesome.github.io/Font-Awesome/](http://fortawesome.github.io/Font-Awesome/) Under **Getting Started**, there are different options for adding the font files. If using the CDN, remember to add the "http" to make it work when you run your page "locally" (on your computer). Simple to use, pick an icon you'd like to use and copy the supplied markup and class and that's it! ###Pseudo Class A CSS pseudo-class is a selector that specifies a specific *state* of the selected element. This lesson uses `:hover` to apply a style only when the user hovers over the element specified by the selector. More about pseudo classes: https://developer.mozilla.org/en-US/docs/Web/CSS/Pseudo-classes
##Home Page Drop down menu 1. Add [Font Awesome's CSS file](http://fortawesome.github.io/Font-Awesome/get-started/) to the `head` of your document and the markup for the [three-bar icon](http://fortawesome.github.io/Font-Awesome/icon/bars/) to the navigation. ``` <link href="http://maxcdn.bootstrapcdn.com/font-awesome/4.2.0/css/font-awesome.min.css" rel="stylesheet"> ``` ```xml <nav> <i class="fa fa-bars" title="menu"></i> <ul> <li><a href="index.html">Home</a></li> <li><a href="about.html">About</a></li> <li><a href="contact.html">Contact</a></li> </ul> </nav> ``` It should look like this: ![](img/workshop/menu-with-icon.png)
##Home Page Drop down menu The next step is to override the horizontal navigation styles that are shared among the other pages and make it specific to the home page. There is already a class of `home` in the body so we can use that to make our selector more specific. 1. Set the list items to display block so they line up on top of each other. ``` .home nav li { display: block; } ``` 1. Right align the icon and the list items. This can be added to the `nav` selector so icon and list items will inherit the styles. Change the font size to make it bigger too. The icon will also need a little bit of padding to make the hover target a little bigger and for alignment. ``` .home nav { text-align: right; font-size: 25px; } .fa-bars { padding: 15px; } ``` 1. Now hide the navigation list and show it when you hover over the icon. Hide the list first. ``` .home nav ul { display: none; } ``` 1. With CSS, you can apply a hover on a parent element and use the descendant selector to change the style of its child elements! ``` .home nav:hover ul { display: block; } .home a:hover { color: black; } ``` Your home page should now look similar to the [homepage-dropdown.html](../project/examples/homepage-dropdown.html) example.
